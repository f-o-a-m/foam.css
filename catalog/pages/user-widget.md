# `User Widget`

By default the User Widget have animation of background-color on interaction.
See`Button--withTransform` and `Button--withShadow` for more animations.

```html
span: 3
dark: true
---
<link rel="stylesheet" href="https://unpkg.com/tachyons@4.9.1/css/tachyons.min.css"/>
<div class="
  User-Widget
  u-padding-m">
  <button class="
    Icon-Widget
    u-border-s
    u-padding-s
    u-color-whitishDim
    u-background-blackish">
    <img src="https://raw.githubusercontent.com/f-o-a-m/foam.css/master/assets/foam_token.png"
    height="20" width="22">
  </button>
  <button class="
    f2 dim br-pill ph4 pv3 ml2 mb2 dib white bg-black
  ">
    0.00 <span class="u-type-s">FST</span>
  </button>
</div>
```
