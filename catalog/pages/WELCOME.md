We use BEM like naming convention:

```
Block { ... }
Block--blockModifier { ... }
Block-element { ... }
Block-element--elementModifier { ... }
```

For utility classes we use naming convention like this:

```
u-utilityName
u-utilityName-utilityVariation
u-utilityName-utilityVariation@condition
```
